F001:Report Change Customer Data
ปัญหาที่พบ
•	เนื่องจาก OC ทำการ Run Report Change Customer Data เพื่อตรวจสอบเคสที่ทำการ Special Change Customer On SSO แล้วระบบดึงข้อมูลที่ทำการเปลี่ยนแปลงข้อมูลลูกค้าผ่านระบบ Mobile Change Customer Type Special (932) มาแสดงด้วย

การแก้ไข
แก้ไขการบันทึกข้อมูลการทำ Request Change ผ่านระบบ Mobile ให้ลงข้อมูลเคส Type Special ให้ถูกต้อง
** ข้อมูลที่ไม่ถูกต้องก่อนการแก้ไขจะไม่ถูกเปลี่ยนแปลง แต่ข้อมูลหลังจากแก้ไขแล้วข้อมูลที่ทำผ่านระบบ Mobile Type Special ต้องไม่แสดงใน Special Report Change Customer **
 
 

F002:Report Key Full Application
 ปัญหาที่พบ
•	เนื่องจาก User ไม่สามารถทำการ Run Report Key Full Application ระบบแสดงข้อความ Data not Found

การแก้ไข
แก้ไขระบบเพื่อให้ Run Report Key Full Application ได้
 

F003:Report Control Key Full Application
 ปัญหาที่พบ
•	เนื่องจาก User ไม่สามารถทำการ Run Report Control Key Full Application ระบบแสดงข้อความ Data not Found

การแก้ไข
แก้ไขระบบเพื่อให้ Run Report Control Key Full Application ได้


F004:Report Change Customer and Inquiry Request
 ปัญหาที่พบ
•	เนื่องจาก User ทำการ Search ข้อมูลลูกค้าหน้า Report Change Customer and Inquiry Request แล้วพบว่า Request No. 23010003166 ไม่แสดงข้อมูล ID No. และ Name-Surname

การแก้ไข
แก้ไขระบบให้แสดงข้อมูล ID No. และ Name-Surname ของ Request No. 23010003166 ที่หน้าจอ Report Change Customer and Inquiry Request
